Git con GitHub

Como usar Git con GitHub

Comandos paso a paso para crear un repositorio local y vincularlo con uno remoto

Para utilizar Git con Github, es necesario seguir algunos pasos. Primero, debes configurar tu nombre y correo electrónico en Git utilizando los siguientes comandos:

git config --global user.name [Tu nombre]

git config --global user.email [Tu correo electrónico]

Luego, debes crear un repositorio local en tu computadora utilizando el comando:

git init

Luego, debes vincular tu repositorio local con un repositorio remoto en Github utilizando el comando:

git remote add origin [URL del repositorio remoto]

Comandos paso a paso para subir o actualizar cambios en el repositorio remoto
